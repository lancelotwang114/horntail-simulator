    怎麼操作

    看攻略

    • ← →　切換上一／下一階段
    • 點下方圓點，或右側階段名稱，直接跳到該階段
    • 右上角可切換 楓谷 / 戰術 / 寶典 三種主題

    排站位（編輯模式）

    • 拖曳職業／標記即可移動位置
    • 方向鍵 微調，按住 Shift 大步移動
    • 選取後右下白點可縮放大小
    • Delete 刪除選取、Ctrl+D 原地複製
    • Ctrl+Z 復原、Ctrl+Shift+Z 重做
    • Esc 取消選取，再按一次離開編輯

    編輯會自動暫存在這台瀏覽器；按「💾 存檔」存成 JSON 長期保存，或「🔗 分享連結」把唯讀畫面傳給隊友。

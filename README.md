# 闇黑龍王 戰術板

楓之谷「闇黑龍王（Horntail）」的**走位戰術板**。在地圖底圖上，用可切換的「階段」呈現各職業站位、目標部位（〇）、已斷部位（✕）、危險／安全／下煙區，幫你和隊友把整套打法視覺化、好溝通。

![預覽](preview.png)

## 線上開啟

👉 <https://lancelotwang114.github.io/horntail-simulator/>

> 這個網址打開是一份**空白起始範本**（保留建議攻略順序的 7 個階段＋固定危險／安全區）。
> 任何人都可以在線上自己排出版本，**不需要、也不會拿到原始 HTML**——大家都透過這個網址製作。

## 功能特色

- 多階段走位：職業圖示在階段間平滑移動，一眼看懂誰該去哪
- 目標標記：〇＝該打的部位、✕＝已斷的部位、💨＝下煙區、自由文字
- 危險／安全區：各有**獨立開關**可隨時開啟／關閉顯示（狀態會記住）
- 內建編輯器：地圖上方工具列**直接拖曳**擺站位與標記，圖上即時微調
- 沿用上一階段站位：一鍵把上一階段的站位帶到這一階段，排整套超省時
- 複製圖片：把目前階段一鍵複製成圖片，直接貼到 LINE／Discord
- 三種視覺主題（楓谷／戰術／寶典）可一鍵切換
- 編輯自動暫存於瀏覽器，並可存成 JSON 檔長期保存、產生唯讀分享連結

## 怎麼看（檢視模式）

- 用左下「上一階段／下一階段」切換，或鍵盤 `←` `→`
- 滑鼠移到下方進度點，可看到該階段名稱；點一下直接跳該階段
- 右側面板顯示該階段的文字說明
- 地圖下方的 🔴 **危險區**／🟢 **安全區** 開關可切換要不要顯示紅／綠框

## 怎麼編（右上切到「✏ 編輯模式」）

- **上方工具列拖放**：地圖上方會出現兩排工具——
  - 「標記」排：目標〇／已斷✕／下煙💨／文字 T
  - 「職業」排：12 個職業圖示
  直接把它們**拖到地圖**放定位即可（同職業可放多個，顯示 ×2、×3）
- **圖上直接編輯**：點地圖上的職業可在其下方打備註；文字方塊可直接打字、用浮動工具列選色；
  選取後拖**右下角白點**調大小、方向鍵或浮動工具列微調位置、複製或刪除
- **階段說明**：右側面板直接打字，行首可切 一般／⚠注意／✓安全，並可用左側握把拖曳排序
- **階段管理**（地圖下方階段條）：點右側大標改階段名稱；可改代表色、`＋新增頁`、`⧉複製頁`、
  `⤵沿用上一階段站位`、`◀上移`／`下移▶`、`🗑刪除頁`、`↻還原範本`
- **頂部工具**：`↩︎/↪︎` 復原重做、`📋` 複製目前階段圖片、`🗂` 下載全部階段圖片、`💾` 存檔、`📂` 讀取、`🔗` 分享

## 做一份自己的版本

1. 打開上面的線上網址（空白範本）。
2. 進「✏ 編輯模式」，照建議順序逐階段排出你的站位與說明。
   - 想以**完整攻略**為起點？按 **「📂 讀取」** 載入本專案附的 `horntail-full-guide.json`，再改成你的版本。
   - 相鄰階段站位差不多時，按 **「⤵ 沿用上一階段站位」** 直接繼承再微調。
3. 按 **「💾 存檔」** 存成你自己的 JSON 檔（例如 `my.json`）。
4. 下次複盤或修改：回到網址 → **「📂 讀取」** 載入你的檔 → 繼續改 → 再「💾 存檔」覆蓋。

> 你的資料都在自己手上的 JSON；要檢視或編輯時，回到這個網址打開即可。
> 「💾 存檔」直接覆蓋檔案需用 Chrome／Edge，其他瀏覽器會自動改成下載。

## 分享給隊友

- **唯讀連結**：編輯模式按 **「🔗 分享」**，把連結傳給隊友，他們打開就是**唯讀檢視**（看不到編輯功能、不能改、標題旁顯示「🔒 唯讀檢視」），畫面跟著你編好的內容。需用線上版（GitHub Pages）才能給別人開。
- **複製圖片**：按 **「📋」** 把目前階段複製成圖片，直接貼到 LINE／Discord（部分瀏覽器不支援複製時會自動改為下載 PNG）。
- **下載全部階段**：按 **「🗂」** 一次把每個階段各存成一張 PNG（檔名含編號與階段名、圖上方加階段標題）。
- 分享連結會自動以 gzip 壓縮（`#vz=`）大幅縮短，舊連結仍可開啟。

## 部署（給維護者）

這是一份**純前端、多檔**網站，直接把以下檔案放到 repo 根目錄、開啟 GitHub Pages 即可：

```
index.html              ← 入口（空白起始範本）
app.js                  ← 程式邏輯與預設範本資料
editor.css              ← 編輯器樣式
directions/             ← base / dir-a / dir-b / dir-c 四個主題樣式
assets/                 ← map.jpg 地圖、jobs/ 職業圖示
horntail-full-guide.json← 完整攻略範例（可在站內「讀取」載入）
preview.png             ← README 預覽圖
LICENSE                 ← MIT 授權
```

> 想換「預設空白範本」的內容，改 `app.js` 最上面的 `STAGES` 陣列即可。

## 攻略鐵則

尾巴一定要「最先」斷，其他部位先斷＝挑戰失敗。

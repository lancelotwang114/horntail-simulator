# 闇黑龍王 互動攻略模擬器（Horntail Simulator）

楓之谷「闇黑龍王（Horntail）」攻略走位的互動模擬器。在地圖底圖上以可切換的「階段」呈現各職業站位、目標部位（〇）、已斷部位（✕）、危險／安全／下煙區，並內建房主編輯器，可自行編排流程後分享給隊友檢視。

## 功能

**檢視模式**
- 多階段切換（上一階段 / 下一階段 / 點圓點跳階段），鍵盤 `←` `→` 操作
- 職業站位在階段間平滑移動，〇目標、✕已斷、危險/安全/下煙區一目了然

**編輯模式（按「✏ 編輯模式」）**
- 拖曳調整職業站位；同職業可放多個分身（顯示 ×2、×3），每個圖示可自訂名字/標籤
- 新增/縮放/刪除 〇✕ 目標（〇可標部位編號 1~8）與下煙區
- 在圖上新增自由文字方塊，或一鍵把「主旨」放到圖上
- 編輯每階段標題、顏色、說明文字
- 階段管理：複製 / 新增 / 刪除 / 前後移；可插入「尾前總站位」參考
- 同一位置超過 3 人會提示（攻略鐵則）
- 自動存稿（localStorage）、匯出/匯入 JSON
- 鍵盤：選取物件後 `方向鍵` 微調、`Esc` 退出

**分享**
- 編輯模式「🔗 複製分享連結」會把整份範本壓進網址，對方打開即**唯讀檢視**（看不到編輯器）
- 需先部署到 GitHub Pages，連結才能給別人開（本機 file:// 只能自己用）

## 使用

直接用瀏覽器打開 `闇黑龍王_互動攻略.html` 即可。需與 `職業/` 資料夾、`default.jpg` 放在一起（圖片走相對路徑）。

## 部署到 GitHub Pages

1. 推上 GitHub repo（`main` 分支）。
2. Settings → Pages → Source 選「Deploy from a branch」→ `main` / `/(root)` → Save。
3. 約 1～2 分鐘後即可從 `https://<帳號>.github.io/horntail-simulator/闇黑龍王_互動攻略.html` 開啟。
   （把主檔改名為 `index.html` 可得到更乾淨的網址 `.../horntail-simulator/`。）

## 檔案結構

| 檔案 | 說明 |
|---|---|
| `闇黑龍王_互動攻略.html` | 主程式（檢視 + 編輯器），走相對路徑 |
| `職業/` | 職業圖示（**必要素材**） |
| `default.jpg` | 地圖底圖（**必要素材**） |
| `闇黑龍王_攻略_單檔.html` | 圖片 base64 內嵌的單檔版（可獨立傳閱，內容為舊版範本） |

## 攻略鐵則

尾巴一定要「最先」斷，其他部位先斷＝挑戰失敗。

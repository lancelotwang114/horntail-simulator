# 闇黑龍王 互動攻略模擬器

楓之谷「闇黑龍王（Horntail）」的**走位攻略互動模擬器**。在地圖底圖上，用可切換的「階段」呈現各職業站位、目標部位（〇）、已斷部位（✕）、危險／安全／下煙區，幫你和隊友把整套打法視覺化、好溝通。

![預覽](preview.png)

## 線上開啟

👉 <https://lancelotwang114.github.io/horntail-simulator/>

## 功能特色

- 多階段走位：職業圖示在階段間平滑移動，一眼看懂誰該去哪
- 目標標記：〇＝該打的部位、✕＝已斷的部位，〇 可標部位編號
- 區域提示：固定的危險區（紅）／安全區（綠），以及各階段的下煙區（灰）
- 內建編輯器：自己排站位、加標記與文字、寫每階段說明
- 編輯自動暫存於瀏覽器，並可存成 JSON 檔長期保存

## 怎麼看

- 用「上一階段／下一階段」切換，或鍵盤 `←` `→`
- 右側（手機在下方）會顯示該階段的說明
- 圖例：〇＝目標、✕＝已斷、紅框＝危險、綠框＝安全、灰框＝下煙

## 怎麼編（按「✏ 編輯模式」）

- **職業**：點職業名加到地圖、拖曳調位置；同職業可放多個（顯示 ×2、×3），點圖示可幫它命名／標籤
- **標記**：加 〇／✕（〇 可標編號）、加下煙區、在圖上加自由文字
- **物件操作**：選取後用右下角把手縮放、方向鍵微調、按刪除移除
- **階段**：改標題與顏色、編每行說明、複製／新增／刪除／調整順序
- 編輯內容會**自動暫存**在這台電腦的瀏覽器；若沒存成檔就關閉會提醒你

## 做一份自己的版本

1. 進編輯模式，從範本改成你的內容（也可從空白頁開始）。
2. 按 **「💾 存檔」** 存成你自己的 JSON 檔（例如 `my.json`）。
3. 下次要複盤或修改：回到本頁 → **「📂 讀取」** 載入你的檔 → 繼續改 → 再「💾 存檔」覆蓋。

> 資料都在你手上的 JSON；要檢視或編輯時，回到這個網頁打開即可。
> 「💾 存檔」直接覆蓋檔案需用 Chrome／Edge，其他瀏覽器會自動改成下載。

## 分享

進編輯模式按 **「🔗 分享連結」**，把連結傳給隊友，他們打開就是**唯讀檢視**（看不到編輯功能、不能改），畫面跟著你編好的內容。分享連結需用線上版（GitHub Pages）才能給別人開。

## 攻略鐵則

尾巴一定要「最先」斷，其他部位先斷＝挑戰失敗。

# 闇黑龍王 互動攻略模擬器

楓之谷「闇黑龍王」攻略走位的互動模擬器。在地圖上以「階段」呈現各職業站位、目標部位（〇）、已斷部位（✕）、危險／安全／下煙區，並可自行編排流程後分享給隊友看。

## 怎麼看

- **上一階段 / 下一階段**：切換各階段的走位
- 鍵盤 `←` `→` 也能切換階段
- 〇＝該打的目標部位，✕＝已斷部位，紅框＝危險區，綠框＝安全區，灰框＝下煙區

## 怎麼編（按「✏ 編輯模式」）

- **拖曳**地圖上的職業圖示調整站位；同職業可放多個（顯示 ×2、×3）
- 點圖示可幫它**取名字／標籤**（例：誘惑左、主坦）
- **加〇 / 加✕**：放目標／已斷標記，〇 可標部位編號；**加下煙**：放下煙區
- **加文字 / 加主旨**：在圖上放說明文字或把階段主旨放上圖
- 選取物件後可**縮放、刪除**，或用**方向鍵微調**、`Esc` 退出
- **階段管理**：複製、新增、刪除、調整順序
- 編輯會**自動儲存**在這台電腦的瀏覽器；也可**匯出 / 匯入 JSON** 備份
- 編好按「✓ 完成編輯」回到乾淨檢視

## 怎麼分享

編輯模式按 **「🔗 複製分享連結」**，把連結傳給隊友，他們打開就是**唯讀檢視**（看不到編輯功能），畫面跟著你編的內容走。
